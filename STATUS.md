## 📝 使用说明

### 当前可用功能

1. **备忘录分析**
   - 输入文本备忘录
   - 上传高驰截图
   - 获得AI分析结果
   - 保存记录

2. **历史记录查看**
   - 查看所有记录列表
   - 查看记录详情

### 启动步骤

1. 安装依赖（见QUICK_START.md）
2. 初始化数据库：`python backend/init_db.py`
3. 启动后端：`python backend/main.py`
4. 启动前端：`cd frontend && npm run dev`

## 🔧 技术细节

### 后端API端点

- `POST /api/memo/analyze` - 分析备忘录
- `POST /api/memo/save` - 保存记录
- `GET /api/records/list` - 获取记录列表
- `GET /api/records/{id}` - 获取记录详情

### 数据库表

- `records` - 主记录表
- `coros_data` - 高驰数据表
- `video_analysis` - 视频分析表（待使用）

## 📌 注意事项

1. **Qwen API配置**：确保config.json中的API Key正确
2. **文件上传**：图片保存在 `uploads/images/` 目录
3. **数据库**：SQLite文件在 `database/running_agent.db`
4. **开发模式**：前端运行在 http://localhost:3000，后端运行在 http://localhost:8000
