## 🚀 启动步骤

### 1. 安装依赖

```bash
# 后端
cd backend
pip install -r requirements.txt

# 前端
cd ../frontend
npm install
```

### 2. 初始化数据库

```bash
cd backend
python init_db.py
```

### 3. 启动应用

**终端1 - 后端：**
```bash
cd backend
python main.py
```

**终端2 - 前端：**
```bash
cd frontend
npm run dev
```

## ✨ 功能说明

### 1. 备忘录分析
- 输入文本备忘录
- 上传高驰截图
- AI自动分析并给出评价
- 保存记录

### 2. 视频分析
- 上传跑步视频
- 选择拍摄角度（侧面/背面）
- MediaPipe提取关键点
- 计算角度/对称性
- 生成可视化图
- AI分析评价
- 保存记录

### 3. 历史记录
- 查看所有记录
- 搜索和筛选
- 查看详情
- 删除记录
- 自动关联同一次跑步

### 4. 数据统计
- 总览指标
- 趋势图表
- 时间范围筛选

### 5. 智能指导
- 本周训练总结
- 趋势分析
- 下周训练建议
- 健康提醒

### 6. 记录关联
- AI自动判断同一次跑步
- 手动关联/解除关联
- 关联记录展示

## 📝 注意事项

1. **首次运行**：MediaPipe首次使用会下载模型，需要一些时间
2. **Qwen API**：确保config.json中的API Key正确
3. **文件路径**：所有上传的文件保存在 `uploads/` 目录
4. **数据库**：SQLite文件在 `database/running_agent.db`
