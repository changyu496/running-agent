## 🚀 启动步骤（3步）

### 1. 安装依赖

```bash
# 后端
cd backend
pip install -r requirements.txt

# 前端
cd ../frontend
npm install
```

### 2. 初始化数据库

```bash
cd backend
python init_db.py
```

### 3. 启动应用

**终端1：**
```bash
cd backend
python main.py
```

**终端2：**
```bash
cd frontend
npm run dev
```
